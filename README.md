# 💫 About Me:
About<br>I'm a software developer and Master of Computer Engineering candidate at the University of Guelph (GPA 87.38/100) with hands-on experience building scalable, cloud-native applications across the full stack.<br> <br>My recent projects include CommuneDrop — a ride and delivery platform with 6+ microservices deployed on AWS EKS using Kubernetes and Terraform — SmartGroc, an AI-powered app with a PySpark data pipeline and OpenAI API integration, and DocuShare, a secure SaaS file-sharing platform built on a multi-tier AWS architecture.<br> <br>I work across Python, JavaScript, TypeScript, React, Node.js, .NET Core, Flask, Apache Kafka, Docker, Kubernetes, and AWS — and I bring strong engineering discipline to everything I build: code reviews, TDD, CI/CD, and clear documentation.<br> <br>Outside of building, I'm a Graduate Teaching Assistant at the University of Guelph where I coach 100+ students in OOP, Data Structures, and Algorithms — conducting structured code reviews and mentoring developers at the start of their careers.<br> <br>I'm currently open to Software Engineer and Software Developer roles across Canada.
